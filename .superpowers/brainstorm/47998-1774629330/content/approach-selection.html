PhysioApp Architecture

Which approach should we go with?

C

React + Vite + Markmap (Recommended)

Best balance of speed-to-ship, Claude Code compatibility, and bundle size. React ecosystem + Markmap for mind maps + vite-plugin-pwa for offline. ~110KB gzipped.

A

React + React Flow (Heavier)

Same React stack but uses React Flow for mind maps. More powerful node editor, but overkill for read-only mind maps. ~150KB gzipped.

B

SvelteKit Static (Lightest)

Smallest bundle (~20KB), pre-rendered HTML. But requires custom D3 mind maps, less Claude Code familiarity, more implementation time.
